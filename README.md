# 💬 Echo Realm – Flutter Chat App

A modern Flutter mobile application for real-time community chatting for a seamless interaction. It leverages Firebase backend services for authentication, storage, and messaging to ensure reliability and scalability.

---

## 📌 Project Overview

**Echo Realm** is a Flutter-based chat application that allows users to:

- ✉️ Sign in with email and password  
- 🆕 Create a new account with optional profile photo  
- 💬 Send and receive messages in real-time  
- 🔔 Receive push notifications for new messages  
- 🎨 Enjoy a custom dark black/red Material 3 UI

## 🛠 Tech Stack

| Layer      | Technologies |
|------------|-------------|
| Framework  | Flutter (Material 3) |
| Language   | Dart |
| Auth       | Firebase Authentication (`firebase_auth`) |
| Database   | Cloud Firestore (`cloud_firestore`) |
| Storage    | Firebase Storage (`firebase_storage`) |
| Messaging  | Firebase Cloud Messaging (`firebase_messaging`) |
| Images     | Image Picker (`image_picker`) |

## 🧱 Core Features

✔ **Email/Password Authentication** — Sign in or register  
✔ **Profile Setup** — Add username and optional photo  
✔ **Realtime Messaging** — Stream messages from Firestore  
✔ **Push Notifications** — Subscribe to chat topic via FCM  
✔ **Modern UI** — Dark black/red theme with custom bubbles  
✔ **Material 3** — Consistent styling across app
